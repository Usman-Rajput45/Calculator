# Calculator

Calculator Project

A simple Calculator web application built with HTML, CSS, and JavaScript. This project is fully responsive, lightweight, and works on both desktop and mobile devices.

Live Demo

Check the live version here: https://tiny-sprinkles-60ca57.netlify.app/

Features
Basic arithmetic operations: addition, subtraction, multiplication, and division.
Clear and delete functionality.
Fully mobile responsive layout.
Clean and minimal UI design.

Technologies Used

HTML5 – Markup structure
CSS3 – Styling and responsive layout
JavaScript – Logic for calculations

Netlify – Deployment

Installation

To run this project locally:

Clone the repository:

git clone https://github.com/Usman-Rajput45/Calculator.git


Navigate to the project folder:

cd Calculator


Open index.html in your browser.

Usage

Click on the buttons to perform calculations.

Press C to clear the screen.

Press DEL to delete the last digit.
